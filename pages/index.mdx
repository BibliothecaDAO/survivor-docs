# Loot Survivor

A roguelike, survival fully onchain game built on Starknet built by Bibliotheca DAO.

## What is Loot Survivor?

Loot Survivor is an onchain arcade machine, you put a small amount of $LORDS, you play and you try to top the leaderboard.
