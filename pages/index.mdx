# Loot Survivor

A roguelike, survival fully onchain game built on Starknet.

Built by Bibliotheca DAO contributors and [@LootRealms](https://twitter.com/LootRealms)community.

## What is Loot Survivor?

Loot Survivor is an onchain arcade machine, you put in a small amount of $LORDS, play and try to top the immutable leaderboard.
