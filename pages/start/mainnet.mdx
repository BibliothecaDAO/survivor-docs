# Getting Started On Mainnet

> This is the mainnet version of Loot Survivor. It costs $3 (<MainnetGamePrice />) to play, along with ETH/STRK to pay for transactions (~$1). We highly reccomend playing your first few games on [Testnet](/start/testnet) for free.

- $3 (<MainnetGamePrice />) per game
- Over 50% of LORDS allocated to top players
- Collectible Beast NFTs
- Uses Verifiable Randomness
- Leaderboard is global & immutable
- Requires Starknet Account

___

## Onboarding Steps

1. Conect a Starknet Account, we recommend using Cartridge Controller for a seamless experience. Loot Survivor also supports Argent and Braavos Accounts.
    - Select the "Login with Cartridge Controller"
    
    <div style={{ display: 'flex', justifyContent: 'center' }}>
        <img src="/getting-started/cartridge-button.png" alt="Cartridge Button" width="250" height="150" />
    </div>

    - Login or Sign Up with your username. If you are signing up, create a pass key using your preffered method to secure your account.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
        <img src="/getting-started/controller-login.png" alt="Cartridge Controller" width="250" height="150" />
    </div>

2. Now that you are connected to the Cartridge Controller you need some ETH for gas and VRF. If you have ETH on Starknet, simply copy the controller address from step 1. For more info on bridging: https://docs.starknet.io/starkgate/overview/.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
        <img src="/getting-started/get-eth.png" alt="Get ETH" width="250" height="150" />
    </div>

3. Finally, the token of the this Arcade Machine is $LORDS. There is a link to AVNU for purchasing LORDS.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
        <img src="/getting-started/get-lords.png" alt="Get LORDS" width="250" height="150" />
    </div>

4. You are ready to go, enjoy playing without needing to sign transactions!
