# Getting Started

New to Loot Survivor? Here is how you can play:

1. Create a Starknet Wallet. Download [ArgentX](https://www.argent.xyz/argent-x/) or [Braavos](https://braavos.app/).
2. Go to [Loot Survivor](https://survivor.realms.world/)
3. Connect your wallet

You can now create an Arcade Account for signature free gameplay. Learn more about [Arcade Accounts](/arcadeaccounts).
