# Getting Started

New to Loot Survivor? Here's how you can play:

1. Get a Starknet Wallet. Download [ArgentX](https://www.argent.xyz/argent-x/) or [Braavos](https://braavos.app/).
2. Go to [Loot Survivor](https://goerli-survivor.realms.world/)
3. Connect your wallet, create an arcade account and play!

Create an Arcade Account for signature free gameplay. Learn more about [arcade accounts](/arcadeaccounts).
