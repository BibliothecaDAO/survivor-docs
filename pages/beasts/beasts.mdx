# Beasts

- There are 75 beasts in Loot Survivor.
- Beasts attack the Head, Chest, Hands, Waist and Feet.
- Beast have dynamic health and attack locations.
- Greatness 19+ beasts unlock special name prefixes.

## Base Damage Calculation

Beast base attack damage is:
`Greatness * (6 - Tier)`

### Beast Attack Types

| Beast     | Weapon   | Armor |
| --------- | -------- | ----- |
| `Magical` | Magic    | Cloth |
| `Hunter`  | Blade    | Hide  |
| `Brute`   | Bludgeon | Metal |
