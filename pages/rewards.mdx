## Lords instant payouts

The contract has a built in distribution model for incentivising different parties. These payouts are distributed as soon as a player pays the LORDS fee to play.

| Party             | Percentage | 
| ----------------- | ---------- |
| `1st Place`       | 27%        |
| `2nd Place`       | 16%        |
| `3rd Place`       | 10%        |
| `Client Provider` | 27%        |
| `Creator`         | 20%        |
